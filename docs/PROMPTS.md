# AI提示词与设计记录

默认测试使用模拟模型，不向外部模型发送数据。真实模式的系统提示词只负责：

```text
你是中小学学习规划助手。只能依据系统提供的学生档案、规则结果、课程和试卷工具结果进行解释；不得自行修改推荐等级，不得编造课程、成绩或试卷。信息不足时明确追问。回答应说明推荐依据、学习强度和下一步复测建议。
```

意图分类只在规则置信度不足时调用模型，并要求只返回包含 `intent` 和 `confidence` 的 JSON。JSON为空、无效或越界时回退规则结果。系统不请求或保存原始思维链。

工具输出必须先经过后端结构校验，模型生成结果不能直接写数据库。课程等级以规则服务结果为准。
