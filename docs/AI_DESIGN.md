# AI、工具、RAG 与记忆设计

系统支持 13 类意图：课程推荐、课程搜索、试卷搜索、学习计划、学情分析、知识问答、学习报告、错题分析、创建订单、我的课程、我的订单、普通对话和未知意图。

编排器执行固定链路：权限隔离 → 会话记忆 → 意图和实体 → 澄清 → 工具 → RAG → Provider → 结构化卡片 → 持久化。八个工具分别负责学生档案、课程推荐、课程搜索、试卷搜索、七天计划、学习报告、错题和订单。

课程等级按真实规则判断：低于70分为基础巩固型，70–89分为中等提升型，90分及以上为拔高拓展型；三个以上薄弱点下调一级；“巩固基础”限定基础型；竞赛/拓展目标只有成绩不低于90且薄弱点不超过一项时允许拔高。

RAG 使用中文字符 2–4 gram TF-IDF 与余弦相似度，默认返回 Top 4。课程、试卷和本地教育资料写入 `rag_documents/rag_chunks`，按来源与哈希去重。完整聊天保存在数据库，模型上下文仅包含结构化摘要和最近20条消息。

DeepSeek 和 Mock 实现相同 Provider 接口。缺少 Key、超时、限流或认证失败时按配置进行安全错误映射或 Mock 兜底。系统不保存原始思维链，也不允许模型覆盖工具返回的 ID、价格、成绩和状态。
