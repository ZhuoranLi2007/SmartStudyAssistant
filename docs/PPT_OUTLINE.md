# 答辩PPT素材

1. 痛点：家长难判断基础、提升或拔高课程，试卷与课程缺少联动。
2. 方案：学生档案 + 可解释规则 + AI工具编排 + 课程/试卷/计划闭环。
3. 架构：HarmonyOS端侧语音/OCR，FastAPI云侧意图、工具、数据库和会话。
4. 创新：推荐不是模型随口生成，规则结果、工具参数和推荐记录均可追踪。
5. 演示：家长注册 → 建档 → 语音咨询 → 推荐提升型 → 加入课程和试卷 → 学生完成 → 家长查看。
6. 安全：密钥留在后端、PBKDF2密码、JWT、家庭隔离和一次性绑定码。
7. AI亮点：13类意图、8个显式工具、中文TF-IDF RAG、会话摘要、结构化卡片与可追踪来源。
8. 可靠性：SSE事件流、相同消息ID幂等降级、Mock兜底和统一错误映射。
7. 测试：ArkTS构建、8项后端自动化测试、异常降级。
8. 边界：当前模型为mock、MySQL及真机AI待最终环境验收，未将云同步宣传为原生分布式。
